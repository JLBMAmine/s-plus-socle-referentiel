# S+ Socle référentiel comptable

Bibliothèque de référence comptable pour le projet S+ — outil hybride méthodologie + plateforme de management stratégique pour PME québécoises.

## Contenu

### Norme IFRS for SMEs
- `ifrs-smes-norme-2025/` — 3e édition de la norme IFRS for SMEs (février 2025, anglais), applicable obligatoirement à compter du 1er janvier 2027

### Taxonomies IFRS XBRL
- `ifrs-taxonomy-2024/` — Taxonomie IFRS Accounting 2024 complète (incluant l'entry point IFRS pour PME)
- `ifrs-taxonomy-2025/` — Taxonomie IFRS Accounting 2025 (full IFRS, sans IFRS-PME — séparé par l'IFRS Foundation depuis 2025)

### IFRS Taxonomy Illustrated (lecture humaine — PDF + Excel)
- `ifrs-iti-2024/` — Documents ITI 2024 (by FS, by Accounting Standard)
- `ifrs-iti-2025/` — Documents ITI 2025 (IAS 1, IFRS 18)

### Versioning
- `ifrs-versioning-2024-2025/` — Rapports HTML de changement entre les taxonomies 2024 et 2025

### Spécifications techniques XBRL
- `xbrl-specs/` — XBRL 2.1, Dimensions 1.0, Technical Considerations, suite de conformance XDT

### Sources canadiennes
- `caseware/` — Codes de liaison Caseware (référence d'inspiration PME québécoises)
- `igrf/` — Index général des renseignements financiers de l'ARC (codes Excel + guide RC4088 PDF et HTML)

## Statut

Dépôt de référence statique. Pas de code applicatif. Sources figées à leurs versions de téléchargement (mai 2026).

Ce dépôt est une **compilation de références** pour le développement du projet S+. Il n'est pas une publication officielle de l'IFRS Foundation, de XBRL International ou de l'ARC.
